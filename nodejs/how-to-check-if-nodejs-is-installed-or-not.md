# How to check if Node.js is installed or not

```bash
node -v
```

- `node -v` - returns installed Node.js version or error (`No such file or directory`) if node is not installed

group: install

## Example: 
```bash
node -v
```
```
-bash: /usr/bin/node: No such file or directory
```
