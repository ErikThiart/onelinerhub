# How to get current date and time

```js
const now = new Date();
console.log(now);
```

- `new Date()` - creates new object to work with dates based on current time (if empty constructor)
- `console.log(now)` - will print given date/time

group: date

## Example: 
```js
const now = new Date();
console.log(now);
```
```
2022-08-11T09:08:17.753Z
```
