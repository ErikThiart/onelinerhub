# Remove item from an array by an index

```javascript
var arr = [1, 2, 3]
arr.splice(1, 1);
```

- `var arr` - declare test array to remove item from
- `(1` - remove second element (indeces start at ```0```)
- ` 1)` - only one element will be removed
