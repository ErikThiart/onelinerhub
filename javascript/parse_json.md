# Parse JSON string

```javascript
var text = '{"a": 1, "b": 2}';
var json = JSON.parse(text);
```

- `var text` - string variable with JSON to parse
- `json` - will contain parsed JSON object
- `JSON.parse` - parses string JSON into an object
