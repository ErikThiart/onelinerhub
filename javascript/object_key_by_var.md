# Access object key by variable

```javascript
var key = 'a';
var obj = {a: 1, b: 2, c: 3};
obj[key] = 11;
```

- `var key` - declare variable with key name
- `var obj` - declare object
- `obj\[key\]` - will access ```a``` property of ```obj``` and write ```11``` to it.
