# How to copy array values (instead of variable reference)

```javascript
var arr = [1, 2, 3, 4, 5];
var copy = arr.slice();
```

- `var arr` - declare test array to copy values from
- `copy` - will contain new array with all values from ```arr```
- `arr.slice()` - will return all values from ```arr```
