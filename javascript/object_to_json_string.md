# Convert object to JSON string

```javascript
var obj = {a: 1, b: 2};
var json = JSON.stringify(obj);
```

- `var obj` - test object to convert to JSON string
- `json` - will contain resulting JSON string
- `JSON.stringify` - converts object in arguments to JSON string
