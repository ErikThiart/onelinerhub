# Convert hex to decimal (integer)

```javascript
var hex = '6A';
var num = parseInt(hex, 16);
```

- `var hex` - declare test HEX number string
- `num` - will contain converted integer
- `parseInt(hex, 16)` - parses ```hex``` string and converts to base-10 integer
