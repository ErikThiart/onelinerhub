# Change image source URL

```javascript
$('img').attr('src', '/new_image.png');
```

- `'img'` - selector for needed image
- `.attr(` - will set specified attribute value
- `'src'` - we're changing ```src``` attribute
- `'/new_image.png'` - URL of a new image to set
