# Get data-* attribute value

```javascript
$('div').data('type');
```

- `'div'` - selector to get needed element
- `.data(` - will return value of specified ```data-*``` attribute
- `'type'` - name of ```data-*``` attribute (assuming following html: ```<div data-type="something">```)
