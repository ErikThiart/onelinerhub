# Batch optimize multiple PNG image

```bash
optipng -o3 /dir/*.png
```

- `optipng` - tool for PNG optimization (```apt install optipng```)
- `-o3` - quality (```0``` - best quality, ```7``` - worst quality)
- `/dir/*.png` - path to multiple PNG image to optimize
