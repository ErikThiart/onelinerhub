# Resize image proportionally

```bash
convert in.jpg -resize 100x out.jpg
```

- `in.jpg` - input image
- `-resize 100x` - will resize an image to 100px width and proportional height (use ```x100``` to resize based on height)
- `out.jpg` - resulting image
